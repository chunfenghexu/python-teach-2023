定义一条文法如下所示 ：
```
<a> ::= ... <b> ... 
或者 
<a> ::+ ... <b> ...
```
::+ 或者 ::= 将文法分为了左部和右部，其中\<a\>表示一个非终结符单词，文法的左部和右部中的非终结符有可能相同，这里就是有可能 `a=b`。
但是如果发生右部中的非终结符与左部非终结符相同，则会导致文法递归产生，这是我们不想要的。
你的任务就是找到这些会导致文法递归产生的文法式，并输出他们的标号。
